 Instruction.hExtract the values of func12_0, func7_5, func3 and opcode_6to2 from the instruction signal (32bits)
To do so, we use bitwise masks : we put a 1 at the bit we want to extract to 1, 0 everywhere else.
If it's a single bit, then we shift right the bit so as to put it in first position.
If not, be carefull that the shift amount is linked to the lowest bit you want to extract.
func12_0 is bit 20, func7_5 is bit 30
func3 is bit 14 to 12 (included) and opcode_6to2 is bit 6 to 2 (included)
