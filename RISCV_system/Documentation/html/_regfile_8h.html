 SC_MODULE (scRegfile)
 

Detailed Description

gather all the elements needed to define the register

Author
Julie Mercier
Version
0.1
Date
2 May 2022

Register File sensitive to the rising edges of the clock.

  • ** INPUTS**

    clock :Falling-edge active clock input
    reset :Active-high synchronous reset input
    w_address, rx_address and ry_address : 5 bits signal (input) because it is all it takes to count until 32
    w_data : 32 bit data input: the data we write in the registers
    w_command,rx_command and ry_command : 1 bit signals to allow the writting (1) or not (0)
    rx and ry : 32 bit data output : the content of register rx and ry
    Local variable : value : array of 32 with the internal value 'value'

  • METHOD SC_THREAD sensitive to reset and positive edges of the clock
  • function void store() : the function that defines the scRegister logic
    Works with an infinite loop and an a wait
    first : if the reset is active at every edge of the clock : for loop to reset all value[i]=0
    If not : if commands are true and that the register's address is not 0 (because no writing allowed in R0), load the corresponding value in wdata and write it in rx or ry if rxcommand and rycommand are set to 1.
    Else, The register takes the value 0.
