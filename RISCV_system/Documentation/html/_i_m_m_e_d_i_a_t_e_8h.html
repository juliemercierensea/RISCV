 SC_MODULE (scIMMEDIATE)
 

Detailed Description

extracts the value of I_imm, U_imm, J_imm, B_imm and S_imm

Author
Julie Mercier
Version
0.2
Date
July 27th 2022
The immediate outputs are signed numbers so that we must be careful to keep them signed according to the value of RI.
To do so, we save the value of RI's MSB in an internal signal and if it's a 1, we set everything unused bit of I, s, U , B or J to 1.Else, we keep them to 0.
