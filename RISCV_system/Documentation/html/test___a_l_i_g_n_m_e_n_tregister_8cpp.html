function called by the main : create the design and run the simulation, produce the vcd file

  • first : definition of all the signals and link to the ports
  • second : start the simulation and create the vcd files
  • third : initialization : we have to test first if the program works as expected when reset is set to 0 and when it is set to 1, the value of wdata is the data we want to write in the selected register

    wadd is the address of the register we will write in (rx or ry but we can't write in both register in the same time)
    ryadd and ryadd are the addresses of the resgiters rx and ry. If their address doesn't match the wadd, nothing changes.

  • 1) no permission at all : if command is set to 0, nothing should change
  • 2) wcommand =1 : we can write
  • 3) rxcommand =1 : we can write in rx. If rxcommand is set to 0, we can't write anything in rx event if wcommand is set to 1
  • 4) rycommand =1 : we can write in ry. If rycommand is set to 0, we can't write anything in ry event if wcommand is set to 1
    Returns
    0
